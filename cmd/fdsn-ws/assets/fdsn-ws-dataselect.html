FDSNWS Dataselect Web Service

The dataselect Web service provides access to earthquake data in miniseed format. Events may be filtered e.g. by network, station, location, channel, and starttime/endtime. The request type is HTTP-GET. Please refer to http://www.fdsn.org/webservice for a complete service description.

Available URLs

  • query
  • queryauth
  • version
  • application.wadl

Feature Notes

  • The result set is limited to 10,000 events. Queries that would return more than 10,000 events receive an HTTP 413 response and will need to be broken in to smaller queries.
